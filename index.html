各授業の学習事項

第1回 WWW と HTML

HTMLについて学んだ。HTMLにおいて、新しい文書を作成する際には<!DOCTYPE html>を1行目に記述し、<html>要素は文書全体を表す。<head>はページに直接表示されない情報を設定し、<meta charset="utf-8">は文字コードを指定する。そして<title>はページのタイトルを設定し、<body>は表示される要素を含む。
演習の成果:演習1-4:HTMLさまざまな要素

第2回 CSS と Webページの装飾


cssについて学んだ。例えば、色を変えたえい文字のフォントを変えたり、グラデーションをつけたり、背景の色を変えたり、プロパティ値に関数を指定することで付箋のように背景の上に新たな文字をかけるスペースなどもできると学んだ。

演習の成果:練習問題２−３

第3回 JavaScript の 基本


javaScriptについて学んだ。javaSCriptはjavaと似た様な名前だが全然違い、javaはプログラミング言語でjavaScriptはスクリプト言語であるという違いがある。また、javaは仮想マシンやブラウザ内で実行されるアプリケーションを作成するが、JavaScriptコードはブラウザ上でのみ実行される。

演習の成果：練習問題3-6

第4回 DOM と JavaScript


cssだけでなくJavaScriptでも文字の色を変えることや、写真を挿入したり、違うサイトのリンクを貼り飛ぶことができるということ学んだ。HTMLとCSSはJavaScriptで対応ができると学んだ。

演習の成果：練習問題４−２〜４−５

第5回 イベント処理


イベントとは、Webページ上で起こった動作や出来事のことを表す。このイベントは、キーボードのキーを押したり、マウスポインタを離したり、合わせたり、マウスボタンでクリックすることで動かすことができると学んだ。

演習の成果：練習5-3

Webコンテンツ作成についての振り返り


私が今回の学習で工夫した点は、シンプルにサイトを作ることです。授業の講義やJavaScriptのサイトを見て、できるだけ無駄を省き、基本に忠実な設計を心がけました。特に、ページのレイアウトやデザインはシンプルで直感的なものにし、ユーザーが簡単に操作できるよう配慮しました。また、JavaScriptのコードも可能な限り簡潔に書き、可読性を高めることで、後から見直した際に理解しやすくなるよう工夫しました。 今回の学習で特に苦心したのは、JavaとJavaScriptの違いを理解することでした。私はこれまでJavaを主に学んでいたため、同じような名前の言語であるJavaScriptにもすぐに慣れるだろうと思っていました。しかし、実際には両者の違いが多く、特に基本的な構文や概念の違いに戸惑いました。 例えば、Javaでのfor-of文とJavaScriptでのfo-ofr文の違いです。 Javaではfor文は次のように書きます。
for (String item : array) { System.out.println(item); }

対してjavaScriptでは

for (let item of array) { console.log(item); }
これらの違いを理解せずに、Javaの構文をそのままJavaScriptに適用しようとしてエラーが発生することが何度かありました。特に、FirefoxのWeb開発ツールでデバッグするときにそのミスに気付くことが多かったです。 今回の学習は非常に楽しかったです。特に、Webの作り方が少しずつ理解できるようになり、自分で作成したWebページが実際に動くのを見ると大きな達成感を感じました。また、学んだことを活かして、これでお金を稼げたら楽しいだろうと思いました。フリーランスのプラグラマーのYOUTUBEでなった経緯や仕事などを見て、自分もそのような働き方ができるようになりたいと強く思いました。 さらに、前回の講義で理解できなかったことが、次の授業で理解できるようになったという経験もありました.例えばcssでの画像挿入はあまり理解できなかったが、次の授業でやったjavaScriptでの画像挿入を理解したらcssでの方も理解できました。実際に自分でコードを書いて実験することで、徐々にその動作を理解することができました。このような一歩一歩の積み重ねが、プログラミングのスキルを向上させる上で非常に重要だと感じました。 また、仲間と協力して問題を解決する楽しさも学びました。プログラミングは時に孤独な作業に感じることもありますが、同じ目標を持つ仲間と知識を共有し合うことで、新しい発見や気づきが生まれます。特に、わからない箇所を人に聞いた際、自分と友達では目をつけるところが違い新たな視点を知れ、プログラミングは人それぞれだと改めて知りました。 総じて、今回の学習を通じて、プログラミングの楽しさと難しさを実感しました。まだまだ学ぶべきことは多いですが、一歩ずつ着実にスキルを身につけていきたいと思います。これからも継続して学び続け、将来的には自分の力でWebアプリケーションを開発し、実際にそのスキルを仕事に活かせるようになりたいです。この経験を糧に、更なる成長を目指して努力していきます。総じて、今回の学習を通じて、プログラミングの楽しさと難しさを実感しました。まだまだ学ぶべきことは多いですが、一歩ずつ着実にスキルを身につけていきたいと思います。 これからも継続して学び続け、将来的には自分の力でWebアプリケーションを開発し、実際にそのスキルを仕事に活かせるようになりたいです。 この経験を糧に、更なる成長を目指して努力していきます。
